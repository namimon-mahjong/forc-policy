# ForC Support / サポート

## 日本語

お問い合わせは以下の方法でご連絡ください。

### 連絡先

- メール: namimon.mahjong@gmail.com
- 件名例: ForC サポート問い合わせ

### お問い合わせ時に記載いただきたい情報

- 端末名（機種名）
- OSバージョン（iOS/iPadOS または Android）
- アプリバージョン
- 問題が発生するまでの手順
- スクリーンショットまたは画面録画（可能であれば）

### よくあるお問い合わせ

- カレンダー権限が許可されていない
- ヘルスデータ権限の設定方法
- 通知設定
- サブスクリプションの購入・復元

## English

If you need help, contact us using one of the following methods.

### Contact

- Email: namimon.mahjong@gmail.com
- Subject example: ForC support request

### What to include in your message

- Device model
- iOS/iPadOS or Android version
- App version
- Steps to reproduce the issue
- Screenshot or screen recording (if available)

### Common help topics

- Calendar permission not granted
- Health data permission setup
- Notification settings
- Subscription purchase and restore
